# План создания сайта (туризм Сочи)

## 1. Цели и требования
- Сайт внушает доверие и демонстрирует возможность организации тура под любой бюджет и срок.
- Чистый HTML (без тяжёлых JS-фреймворков на клиенте), с возможностью масштабирования.
- Максимальная SEO-готовность для Яндекса.
- Все заявки из форм отправляются в Telegram.
- Контакты:
  - Телефон: 89138907262
  - Адрес: ул. Куйбышева, 33
- Формы заявок: **Имя, Бюджет, Кол-во участников, Телефон, Удобное время звонка**.
- Каждая услуга и каждый тур — отдельная страница.

## 2. Этапы работ

### Этап 1. Каркас и базовая архитектура
1. **Статическая архитектура** (SSG):
   - Генерация HTML на сборке.
   - Быстрая индексация и стабильность.
2. **Структура каталогов**:
   - `/pages` — страницы (каждая услуга/тур отдельной страницей).
   - `/components` — переиспользуемые блоки.
   - `/styles` — глобальные стили.
   - `/public` — статические ресурсы.
3. **Базовые страницы**:
   - Главная
   - О компании
   - Контакты
   - Страница благодарности (после отправки формы)
4. **SEO каркас**:
   - `title` 50–60 символов
   - `description` 140–160 символов
   - Canonical URL
   - Open Graph теги

### Этап 2. Семантика, дизайн, доверие
1. **Семантика HTML5**:
   - `header`, `nav`, `main`, `section`, `article`, `footer`.
2. **Блоки доверия**:
   - Реальные фото/видео
   - Отзывы
   - Гарантии и прозрачные условия
3. **Продающая структура**:
   - Hero с сильным УТП
   - Категории услуг
   - Примеры туров
   - Блоки с жильём
   - Блок с видео-услугами
   - CTA формы

### Этап 3. SEO и индексация (Яндекс)
1. `robots.txt` и `sitemap.xml`.
2. Настройка **Яндекс.Метрики**.
3. Верификация **Яндекс.Вебмастера**.
4. Оптимизация скорости (LCP < 2.5s).

### Этап 4. Интеграции
1. **Telegram-бот**:
   - Передача всех полей формы.
   - Указание источника страницы.
2. Валидация формы:
   - Телефон: обязательный
   - Бюджет: число

### Этап 5. Масштабирование
1. Возможность добавления новых туров/услуг.
2. Единый шаблон для карточек услуг.
3. Возможность масштабировать под другие регионы.
