# Разнорабочие Сочи

Веб-сайт для сервиса предоставления разнорабочих в Сочи. Создан на Next.js с упором на производительность, SEO и удобство использования.

## Особенности

- ✅ **Адаптивный дизайн** - работает на всех устройствах
- ✅ **SEO-оптимизация** - структурированные данные, Open Graph, правильные meta теги
- ✅ **Быстрая загрузка** - статическая генерация страниц
- ✅ **Мобильное меню** - удобная навигация на смартфонах
- ✅ **Рабочие формы** - интеграция с WhatsApp
- ✅ **Доступность** - семантичная разметка, ARIA атрибуты

## Технологии

- **Next.js 14.2.3** - React фреймворк
- **React 18.2.0** - UI библиотека
- **CSS3** - стилизация с кастомными свойствами

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка продакшн версии
npm run build

# Запуск продакшн сервера
npm start

# Проверка ESLint
npm run lint
```

## Структура проекта

```
├── pages/              # Страницы сайта (Pages Router)
│   ├── _app.js        # Основной layout приложения
│   ├── _document.js   # HTML документ с lang атрибутом
│   ├── index.js       # Главная страница
│   ├── uslugi.js      # Страница услуг
│   ├── ceny.js        # Страница цен
│   ├── kontakty.js    # Страница контактов
│   ├── o-nas.js       # О компании
│   ├── otzyvy.js      # Отзывы
│   ├── portfolio.js   # Портфолио работ
│   └── blog/          # Блог
├── components/        # Переиспользуемые компоненты
│   ├── ContactForm.js # Форма обратной связи
│   └── JsonLd.js      # Компонент для Schema.org разметки
├── styles/           
│   └── globals.css    # Глобальные стили
└── public/           # Статические файлы
    ├── favicon.svg    # Иконка сайта
    ├── robots.txt     # Настройки для поисковых роботов
    └── sitemap.xml    # Карта сайта
```

### 2. Контактные данные

- Адрес: ул. Ленина, 129, Адлер, Краснодарский край

### 3. Переменные окружения

Созданы файлы:
- `.env.production` - содержит `NEXT_PUBLIC_SITE_URL=https://prorabov.ru`
- `.env.local.example` - пример для локальной разработки

### 4. Аналитика (опционально)

Добавьте в `pages/_app.js` или `pages/_document.js`:
- Яндекс.Метрика
- Google Analytics

### 5. Backend для форм (рекомендуется)

Сейчас формы отправляют данные через WhatsApp. Для продакшна рекомендуется:
- Настроить отправку на email
- Интегрировать с CRM системой
- Добавить капчу от ботов
